Quem sou eu?

Olá, meu nome é Matheus de Paula Azevedo, tenho 19 anos e moro em São Paulo, Brasil. Sou estudante de TI e estou no primeiro semestre de Análise e Desenvolvimento de Sistemas.

No fim de 2024, iniciei meus estudos em Python, mas após refletir, vi que precisava de uma base sólida em desenvolvimento web. Reiniciei minha jornada com HTML, CSS e JavaScript, e agora me esforço diariamente para consolidar esse conhecimento e alcançar minha carreira na área.

Decidi criar este projeto para aprender HTML e CSS na prática, mergulhando de cabeça no código e colocando a mão na massa!

Lembrando que Este projeto é apenas um experimento para aprendizado — não tem a pretensão de ser um site profissional. Estou aqui para errar, testar e absorver o máximo de HTML e CSS no processo!
